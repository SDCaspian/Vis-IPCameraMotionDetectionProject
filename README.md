# Vis-IPCameraMotionDetectionProject
This program is built on the open-source Android app IP Camera by Chinese developer Shen Yao and OpenCV. It supports LAN real-time video streaming and motion detection recording, letting users turn spare Android devices into low-cost surveillance cameras.

## How to use？
First, please download the 'IP Camera' app from (https://nm-sywd.iddadnn.com/app/202307/ipcamera_66691.apk?tk=eJIoNAIQOEOIOkNcIMOIIAOQOgIMOAIYOJcNbVY8NYM5csLJIoMgNcM0y3jxTsnij3D3T3Tsmijsnij0TsnijsmiipGhWyV2j5ShGiC6j1jxj33=), but it's in Chinese, so please use translation software.After turning on the camera service in the software, record the camera's IP, username, and password.
Then, please use a Python editor to modify your camera information (or enter it after running) (Note: some libraries need to be installed by yourself), and run it to remotely view the camera feed and save motion detection.
